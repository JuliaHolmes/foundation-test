CSS Box Model

CSS Box Model is helpful in defining the borders of elements. Use border-box to define your elements size based on the border. The opposite would be using content-box which ignores the border when defining size.

In this section of the course, we get into more detail of how to structure typography using CSS. Typography is the way that text is arranged and presented.

Float Characteristics

Floats are best used to align content within a container. You can align left or right. It's important to use display: flow-root; in your css for a container in order for overflow content to be included within a container that has a floating object within it.

Flexbox

Flexbox is a way of defining flow of content in CSS. Whether it's in rows or columns, you can set things to wrap and define how much space certain child items take up within the container. Spacing can be set using the gap attribute or setting align-content attribute.
