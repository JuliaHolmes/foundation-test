Floats

Float Left

It's best to apply margin around the block element and not the text in these instances. Be careful of using 'clear' to to end floating sections, it will not contain previous float divs.
The proper way to get around this is to use display: flow-root; in a section so that the container goes around all of the content it contains and additional sections are bumped down from a floating object.

In the past floats were used to create columns on a page. Now with modern CSS we have other methods available, including Display Flex and Display Grid

Floats are helpful for floating items within a container, just make sure you use flow-root to have a container encompass all overflowing content.
